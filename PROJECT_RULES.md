# 金属材料科研训练知识库网站：项目长期规则

## 1. 项目名称

推荐项目名称：

```text
金属材料科研训练知识库
```

英文项目目录名：

```text
research-knowledge-site
```

网站副标题：

```text
面向本科生科研训练的金属材料知识体系、实验技能与项目实践记录
```

## 2. 网站定位

本网站是一个个人科研训练知识库，不是普通博客，也不是单纯展示页面。

网站的核心目标是：

1. 系统整理金属材料科研训练所需知识；
2. 降低后续查资料成本；
3. 建立由浅入深的学习路径；
4. 服务实验室入门、科研训练、保研积累和项目汇报；
5. 支持长期更新和复习。

## 3. 第一版网站边界

第一版只完成静态知识库。

第一版必须完成：

1. 首页；
2. 学习路线图；
3. 课程化目录；
4. 一级模块页面；
5. 基础文章页面；

7. 本地搜索；

## 6. Markdown 文章头部规范

每篇正式文章建议使用 frontmatter：

```markdown
---
title: 文章标题
description: 一句话说明本文解决什么问题
level: Level 1
category: 金属材料基础课
tags:
  - 金属材料
  - 科研训练
updated: 2026-05-28
---
```

字段说明：

* title：文章标题；
* description：简短说明文章作用；
* level：难度等级；
* category：所属课程模块；
* tags：检索标签；
* updated：最后更新时间。

## 7. 难度分级规范

### Level 0：科研常识

适合刚接触科研训练时了解。

特点：

* 不要求深入推导；
* 主要建立基本认识；
* 能听懂老师和学长说什么即可。

### Level 1：本科基础

必须理解。

特点：

* 属于材料专业基础知识；
* 后续实验和论文阅读会频繁用到；
* 需要能用自己的话解释。

### Level 2：实验入门

需要能按流程操作。

特点：

* 涉及实验步骤；
* 涉及设备使用；
* 涉及样品制备；
* 需要掌握注意事项和常见错误。

### Level 3：科研分析

需要能解释现象。

特点：

* 不只知道结果，还要知道原因；
* 能把组织、性能、表征结果联系起来；
* 能用于组会汇报。

### Level 4：项目应用

需要能用于科研表达。

特点：

* 可用于论文、PPT、答辩、简历；
* 能形成完整证据链；
* 能进行批判性分析。

## 8. 文章正文规范

每篇知识文章不应只写定义，而应说明：

1. 这个知识点为什么重要；
2. 它解决什么科研问题；
3. 基本原理是什么；
4. 在金属材料研究中怎么用；
5. 常见误区是什么；
6. 与其他知识点有什么关系；
7. 学到什么程度才算够；
8. 可以用哪些问题检查自己是否掌握。

## 9. 实验类文章规范

实验类文章必须包含：

1. 实验目的；
2. 适用材料或场景；
3. 所需设备；
4. 操作流程；
5. 关键参数；
6. 安全注意事项；
7. 常见失败原因；
8. 结果判断标准；
9. 实验记录模板；
10. 复习问题。

实验类文章不得写成泛泛科普。

## 10. 表征类文章规范

表征方法类文章必须包含：

1. 该方法能看什么；
2. 该方法不能看什么；
3. 基本原理；
4. 样品要求；
5. 常见图像或图谱；
6. 图像或图谱怎么看；
7. 常见误判；
8. 与其他表征方法的互证关系；
9. 在金属材料科研中的用途；
10. 掌握程度要求。

## 11. 项目实践类文章规范

项目实践类文章必须区分：

1. 已确认事实；
2. 学长或老师给出的要求；
3. 自己的理解；
4. 待验证问题；
5. 后续计划。

不得把猜测写成确定结论。

建议使用如下标记：

```markdown
> 已确认：
> 待确认：
> 我的理解：
> 后续问题：
```

## 12. 复习卡片规范

复习卡片应短小、清晰、可背诵。

建议格式：

```markdown
## 问题：什么是晶粒？

### 简答

晶粒是多晶材料中取向基本一致的小晶体区域。

### 关键点

- 晶粒之间由晶界分隔；
- 晶粒尺寸会影响强度和塑性；
- 晶粒可以通过 OM、SEM、EBSD 等方法观察或分析。

### 常见误区

不要把晶粒和相直接等同。
```

## 13. 内容可信度规则

内容必须区分：

1. 教材基础知识；
2. 课堂知识；
3. 文献结论；
4. 实验事实；
5. 个人理解；
6. 暂时推测。

如果没有确切来源，不要写成绝对结论。

可以使用：

```text
可能
通常
在多数情况下
需要结合具体材料体系判断
需要进一步实验验证
```

不得滥用：

```text
一定
必然
完全证明
绝对说明
```

## 14. 图表与图片规则

所有图片必须有：

1. 文件名；
2. 图题；
3. 来源说明；
4. 是否原创；
5. 是否需要后续替换；
6. 简短解释。

不得直接堆图片。

图片附近必须说明：

1. 图中看什么；
2. 为什么看这个；
3. 图与知识点有什么关系；
4. 容易误读的地方。

3. 没有解释科研用途；
4. 没有常见误区；
