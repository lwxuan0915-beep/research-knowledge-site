# 金属材料科研训练知识库网站：第一版任务清单

## 当前阶段

当前阶段：第一版网站工程化启动阶段。

已完成：

* 已确定网站定位；
* 已确定知识体系大纲；
* 已确定一级目录结构；
* 已确定采用文档站路线；
* 已确定第一版不做复杂后端功能。

当前目标：

建立一个可以本地运行、结构清晰、便于长期维护的金属材料科研训练知识库网站。

## 第一版技术选择

第一版默认采用：

```text
VitePress + Markdown
```

原因：

1. 适合文档型知识库；
2. Markdown 维护成本低；
3. 适合课程化目录；
4. 支持侧边栏；
5. 支持搜索；
6. 支持深色模式；
7. 适合静态部署；
8. 不需要数据库。

## 总体任务顺序

第一版任务必须按以下顺序推进：

```text
任务 0：只读项目并确认理解
任务 1：初始化 VitePress 项目
任务 2：建立目录结构
任务 3：配置首页、导航栏和侧边栏

任务 7：补充 README
任务 8：运行构建验证
任务 9：输出第一版验收报告
```

不得跳过任务 0。

不得一次性执行所有任务。
