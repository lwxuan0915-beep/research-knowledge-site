# 实验记录与周报模板

实验记录和周报是本科生科研训练中最容易被低估的能力。

很多同学刚进课题组时，会觉得实验记录只是“把今天做了什么写下来”，周报只是“每周给老师交个总结”。这种理解太浅。

真正有价值的实验记录和周报，应该具备三个作用：

```text
可复查：以后能知道实验到底怎么做的
可分析：能从记录中找到现象、问题和证据
可积累：能逐步沉淀成组会汇报、论文段落和保研科研经历
```

金属材料科研训练尤其依赖记录，因为样品状态、工艺参数、制样细节、表征条件、图像编号和数据路径一旦混乱，后续分析就会非常困难。

---

## 1. 本文学习目标

学完本文后，应能做到：

1. 理解实验记录和周报在科研训练中的作用；
2. 区分流水账式记录和科研型记录；
3. 掌握金属材料实验记录应包含的基本信息；
4. 能规范记录样品编号、工艺参数、测试条件和原始数据路径；
5. 能把每周实验结果整理成“问题—工作—结果—分析—下一步”的周报；
6. 能把周报转化为组会汇报、项目总结和保研科研经历；
7. 避免实验做完但数据无法复查、结果无法解释、周报没有逻辑的问题。

---

## 2. 为什么实验记录非常重要

金属材料实验通常具有很强的过程依赖性。

同一个样品，如果出现以下任何信息缺失，后续结果都可能无法解释：

* 样品编号不清；
* 热处理温度或时间记错；
* 腐蚀溶液浓度没记录；
* 金相腐蚀时间没记录；
* SEM 观察区域没记录；
* EDS 点扫位置没对应图片；
* XRD 测试角度范围没记录；
* 拉伸试样尺寸没记录；
* 原始数据路径找不到；
* 图像文件名和样品状态对不上。

这些问题一旦出现，后面再补救会非常困难。

科研训练不是只看“最后有没有图”，而是看：

```text
实验过程是否可追溯
数据来源是否可复查
结论是否有证据支撑
```

## 3. 实验记录和周报的区别

实验记录和周报不是同一个东西。

| 项目   | 实验记录             | 周报                     |
| ---- | ---------------- | ---------------------- |
| 时间尺度 | 每次实验、每天记录        | 每周总结                   |
| 重点   | 过程细节和原始信息        | 阶段进展和逻辑整理              |
| 作用   | 保证可复查            | 保证可汇报                  |
| 内容   | 样品、参数、仪器、现象、数据路径 | 本周目标、完成内容、结果分析、问题、下周计划 |
| 风格   | 细致、客观、完整         | 结构化、逻辑清楚、突出重点          |
| 读者   | 未来的自己、同组成员       | 老师、学长、组会听众、未来材料整理者     |

简单说：

```text
实验记录负责“留下证据”
周报负责“整理逻辑”
```

两者缺一不可。

---

## 4. 低质量记录是什么样的

很多初学者的实验记录会写成：

```text
今天做了金相。
今天拍了 SEM。
今天测了 XRD。
今天数据有点问题。
下周继续做。
```

这类记录最大的问题是：以后无法复查。

它没有说明：

* 做的是哪个样品；
* 样品状态是什么；
* 具体工艺参数是什么；
* 用了什么设备；
* 观察区域在哪里；
* 得到了什么现象；
* 原始数据存在哪里；
* 出现了什么异常；
* 下一步为什么这样安排。

这种记录即使写了，也很难帮助科研推进。

---

## 5. 高质量实验记录的基本原则

实验记录应遵循五个原则。

### 5.1 及时记录

实验当天就记录，不要等几天后凭记忆补。

很多细节很容易忘，例如：

* 腐蚀时间；
* 抛光状态；
* 样品是否掉边；
* SEM 观察区域；
* 仪器报错；
* 数据异常；
* 老师或学长临时提醒。

### 5.2 原始记录优先

实验记录不是为了好看，而是为了真实。

如果实验出现问题，要如实记录：

```text
样品边缘有崩口
腐蚀后局部过腐蚀
SEM 图像有充电现象
XRD 背景噪声较高
拉伸断裂位置不在标距中部
```

不要为了“显得顺利”而删掉异常信息。

异常往往是后续分析和改进的关键。

### 5.3 样品编号必须统一

样品编号是科研记录的核心。

同一个样品在不同环节必须保持一致：

```text
制样编号
热处理编号
金相图片编号
SEM 图片编号
EDS 文件编号
XRD 数据编号
拉伸数据编号
周报编号
```

如果编号不统一，后续几乎无法建立证据链。

### 5.4 参数必须具体

不能写：

```text
高温处理一段时间
腐蚀了一会儿
放大倍数比较高
拉伸速度正常
```

应写成：

```text
固溶处理：520 ℃ × 2 h，水冷
时效处理：180 ℃ × 8 h，空冷
腐蚀条件：3.5 wt.% NaCl 溶液，室温，24 h
SEM：二次电子模式，放大 2000×，加速电压 15 kV
拉伸：标距 25 mm，拉伸速率 1 mm/min
```

科研记录必须具体到别人能大致复现。

### 5.5 数据路径必须清楚

每次实验记录都应写清楚原始数据位置。

例如：

```text
原始图片路径：D:\research-data\2026-05-30\SEM\A1
XRD 原始数据：D:\research-data\2026-05-30\XRD\B2.raw
拉伸数据：D:\research-data\2026-05-30\tensile\C3.csv
处理后图片：D:\research-data\processed\2026-05-30
```

不要只把图片粘进 Word 或 PPT，而不保存原始文件。

## 6. 金属材料实验记录应包含什么

每次实验记录至少包含以下信息。

### 6.1 基础信息

```text
日期：
记录人：
项目名称：
实验目的：
样品编号：
样品状态：
实验地点：
使用仪器：
参与人员：
```

样品状态要写清楚，例如：

```text
未处理态
固溶态
时效态
冷轧态
退火态
腐蚀 24 h 后
焊接接头热影响区
```

### 6.2 实验目的

实验目的不能写得太空。

低质量写法：

```text
做 SEM。
```

更好的写法：

```text
通过 SEM 观察腐蚀后样品表面形貌，判断是否存在点蚀坑、裂纹或第二相周围优先腐蚀现象。
```

再如：

```text
通过 XRD 判断热处理后样品是否出现新的物相或峰位变化，为后续组织—性能分析提供相结构证据。
```

### 6.3 实验条件

不同实验需要记录不同条件。

金相制样应记录：

* 切割方式；
* 镶嵌方式；
* 砂纸目数；
* 抛光剂；
* 腐蚀剂；
* 腐蚀时间；
* 观察倍数。

SEM/EDS 应记录：

* 样品表面状态；
* 是否喷金；
* 加速电压；
* 工作距离；
* 探测器模式；
* 放大倍数；
* 点扫、线扫或面扫位置；
* 图片编号。

XRD 应记录：

* 扫描角度范围；
* 步长；
* 扫描速度；
* 靶材；
* 电压电流；
* 样品表面状态；
* 测试面方向。

拉伸测试应记录：

* 试样尺寸；
* 标距；
* 横截面积；
* 拉伸速率；
* 测试温度；
* 断裂位置；
* 是否有效断裂。

腐蚀实验应记录：

* 溶液成分；
* 浓度；
* pH；
* 温度；
* 浸泡时间；
* 样品暴露面积；
* 是否除氧；
* 电化学测试参数；
* 腐蚀后清洗方式。

### 6.4 实验现象

实验现象要客观描述。

例如：

```text
样品 A1 抛光后表面存在轻微划痕，边缘有少量崩口。
腐蚀 30 s 后晶界显示较清楚，但局部区域出现过腐蚀。
SEM 观察发现样品表面存在尺寸不均匀的腐蚀坑，部分腐蚀坑靠近颗粒状第二相。
XRD 图谱背景噪声较高，低角度区域存在轻微起伏。
拉伸样品 C2 断裂位置靠近夹持端，该数据需要谨慎使用。
```

不要只写“正常”“还可以”“不太好”。

### 6.5 初步判断

初步判断要和证据绑定。

例如：

```text
初步判断腐蚀坑可能与局部组织不均匀有关，但目前只有 SEM 形貌证据，仍需结合 EDS 面扫和更多区域观察验证。
```

或者：

```text
硬度升高可能与时效后析出相形成有关，但目前缺少 TEM 或 XRD 证据，不能直接下结论。
```

## 8. 周报应该解决什么问题

周报不是实验记录的简单复制。

周报要回答五个问题：

```text
本周原本要解决什么问题？
本周实际完成了什么工作？
结果说明了什么？
现在遇到什么问题？
下周准备怎么推进？
```

因此，一份合格周报应包括：

1. 本周目标；
2. 本周完成工作；
3. 关键结果；
4. 初步分析；
5. 遇到的问题；
6. 下周计划；
7. 需要老师或学长帮助的问题；
8. 附件或数据路径。

---

## 9. 低质量周报与高质量周报的区别

### 9.1 低质量周报

```text
本周完成了金相制样和 SEM 观察。
下周继续做 XRD 和拉伸测试。
```

这个周报太空，老师无法判断你的进展。

### 9.2 高质量周报

```text
本周围绕“退火温度对样品组织和硬度的影响”开展实验。完成了 A1、A2、A3 三组样品的金相制样和初步显微观察。结果显示，随着退火温度升高，晶粒尺寸有增大趋势，A3 样品局部出现晶粒粗化现象。硬度初步测试显示 A3 样品硬度低于 A1 和 A2，可能与回复、再结晶和晶粒长大导致位错密度降低有关。目前问题是金相腐蚀时间不够稳定，部分区域存在过腐蚀。下周计划优化腐蚀时间，并补充 EBSD 或更多金相区域统计，以验证晶粒尺寸变化是否具有代表性。
```

这个版本包含：

* 研究问题；
* 完成工作；
* 结果现象；
* 机制解释；
* 当前问题；
* 下周计划。

这才是科研型周报。

## 11. 周报写作的核心逻辑

每一份周报都应该遵循：

```text
目标
→ 工作
→ 结果
→ 分析
→ 问题
→ 计划
```

不要写成：

```text
做了 A
做了 B
做了 C
下周继续
```

更好的写法是：

```text
为了回答 ______ 问题，本周完成了 ______。
结果显示 ______。
这可能说明 ______。
但目前仍存在 ______ 不确定性。
所以下周准备 ______。
```

这套表达以后可以直接用于组会汇报。

---

## 12. 实验结果怎么写

写实验结果时，要尽量避免主观词。

低质量写法：

```text
效果不错。
图比较清楚。
数据还可以。
样品有点问题。
```

更好的写法：

```text
A1 样品晶界显示较清楚，但局部区域存在过腐蚀。
B2 样品 SEM 图像中可见颗粒状第二相，主要分布在晶界附近。
C3 样品拉伸曲线在初始阶段存在异常弯曲，可能与夹持滑移或引伸计问题有关。
XRD 图谱中未观察到明显新峰，但主峰存在轻微宽化趋势。
```

实验结果要做到：

```text
具体
客观
可复查
能对应图表或数据
```

## 13. 初步分析怎么写

初步分析不能把不确定结论写死。

### 13.1 证据不足时

可以写：

```text
该现象可能与 ______ 有关。
目前只能初步推测 ______。
仍需通过 ______ 进一步验证。
```

### 13.2 证据较充分时

可以写：

```text
结合 ______ 和 ______ 结果，可以认为 ______ 是影响该现象的重要因素之一。
```

### 13.3 不要写

```text
完全证明 ______。
唯一原因是 ______。
必然导致 ______。
已经彻底解决 ______ 问题。
```

周报是阶段性记录，不是最终论文结论。结论强度必须克制。

---

## 14. 如何写遇到的问题

遇到问题不是丢脸，写不清问题才是真问题。

低质量写法：

```text
SEM 不太好。
XRD 有问题。
样品坏了。
数据不对。
```

更好的写法：

```text
SEM 图像在高倍下出现局部模糊，可能与样品表面抛光质量不足或充电现象有关。下一步计划重新抛光并尝试调整加速电压。
```

或者：

```text
XRD 图谱背景噪声较高，可能与样品表面不平整或测试时间较短有关。下一步计划重新制样，并适当提高扫描时间。
```

问题记录应包括：

```text
问题是什么
可能原因是什么
影响了什么结果
准备如何解决
需要谁帮助
```

---

## 15. 如何写下周计划

下周计划不能只写“继续做”。

低质量写法：

```text
下周继续做 SEM 和 XRD。
```

更好的写法：

```text
下周计划补充 A1、A2、A3 三组样品的 SEM 观察，重点比较第二相分布和腐蚀坑位置之间的关系，以验证局部腐蚀是否与组织不均匀有关。
```

一个合格计划应包含：

* 做什么；
* 做哪个样品；
* 为什么做；
* 预期输出是什么；
* 完成后能回答什么问题。

## 16. 图表和附件如何整理

周报中的图表不需要堆太多，但关键图必须清楚。

### 16.1 图片命名

建议使用：

```text
日期_样品编号_方法_状态_序号
```

例如：

```text
2026-05-30_A1_SEM_corrosion-24h_001
2026-05-30_A1_EDS_mapping_corrosion-24h_001
2026-05-30_B2_XRD_aged-180C-8h
2026-05-30_C3_tensile_curve
```

### 16.2 图注写法

低质量图注：

```text
SEM 图。
```

更好的图注：

```text
图 1 A1 样品腐蚀 24 h 后的 SEM 表面形貌。可见局部腐蚀坑主要分布于颗粒状第二相附近，提示该区域可能存在局部腐蚀敏感性。
```

### 16.3 数据路径

周报最后应写：

```text
原始数据路径：
处理后图片路径：
表格路径：
本周文献笔记路径：
```

这样以后回头找数据会轻松很多。

## 17. 周报如何服务组会汇报

周报写得好，组会汇报会简单很多。

周报可以直接转成组会结构：

```text
本周目标
→ 本周实验
→ 关键结果图
→ 结果分析
→ 当前问题
→ 下周计划
```

组会中不需要把所有细节都讲完，但必须讲清楚逻辑。

组会表达模板：

```text
本周主要围绕 ______ 问题开展工作。
我们完成了 ______ 实验，涉及 ______ 样品。
结果显示 ______。
从材料学机制上看，这可能与 ______ 有关。
但目前证据仍不足，主要问题是 ______。
下一步计划通过 ______ 进一步验证。
```

## 18. 周报如何服务保研科研经历

保研材料和面试中，科研经历不能只写“参与实验”。

如果你长期保留高质量周报，就可以整理出：

* 项目背景；
* 自己承担的任务；
* 掌握的实验技能；
* 得到的阶段性结果；
* 遇到的问题；
* 解决问题的方法；
* 对材料学机制的理解；
* 后续改进方向。

低质量表达：

```text
参与金属材料课题，掌握 SEM、EDS、XRD。
```

更好的表达：

```text
参与某金属材料组织与性能关系研究，主要负责样品制备、金相观察和 SEM/EDS 表征。通过每周实验记录和结果分析，逐步建立了从工艺参数、显微组织到性能变化的证据链，能够对第二相分布、腐蚀形貌和力学性能之间的关系进行初步解释。
```

这就是周报长期积累的价值。

## 19. 常见误区

### 误区 1：周报写成流水账

只写“做了什么”，不写“为什么做”和“结果说明什么”。

### 误区 2：没有样品编号

样品编号不清，后续所有数据都可能失去价值。

### 误区 3：只保存结果图，不保存原始数据

处理后的图不能替代原始数据。

### 误区 4：问题写得太模糊

“数据不对”“图不好”不是有效问题描述。

### 误区 5：下周计划没有目标

“继续做实验”不是计划。计划必须说明要回答什么问题。

### 误区 6：把猜测写成结论

周报中的分析常常是初步判断，必须保留不确定性。

### 误区 7：不记录异常

异常信息可能是解释结果的关键，不应该省略。

### 误区 8：只给老师看结果，不暴露问题

老师和学长需要知道你的问题，才能帮你调整方向。

## 20. 本科生应掌握到什么程度

### 20.1 最低掌握要求

至少应能做到：

1. 每次实验后当天完成记录；
2. 样品编号和实验参数清楚；
3. 原始数据路径可追溯；
4. 周报能说明本周做了什么；
5. 能记录遇到的问题和下周计划。

### 20.2 科研训练要求

进入课题组后，应进一步做到：

1. 每份周报都有明确研究问题；
2. 能把实验结果和材料机制联系起来；
3. 能说明证据是否充分；
4. 能提出下一步验证方案；
5. 能把周报转化为组会汇报。

### 20.3 保研科研经历要求

如果希望把科研训练用于保研，应达到：

1. 能从多周周报中提炼项目主线；
2. 能讲清楚自己承担的任务和贡献；
3. 能说明自己如何处理实验问题；
4. 能展示持续推进科研任务的能力；
5. 能把记录转化为简历、面试和项目总结材料。

## 23. 发给老师或学长的简短汇报模板

有时候不需要完整周报，只需要发一段简短消息。

可以使用：

```text
老师/学长您好，我这周主要完成了 ______ 工作。结果上看，______ 样品出现了 ______ 现象，初步判断可能与 ______ 有关。目前主要问题是 ______，我计划下周通过 ______ 进一步验证。相关原始数据和图片已整理在 ______ 路径下，想请您帮忙看一下 ______ 是否合理。
```

```text
老师我做完了，您看一下。
```

要专业很多。
