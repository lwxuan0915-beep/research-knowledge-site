# 文献阅读与科研写作

本模块用于训练金属材料科研中的两项基础能力：

```text
读懂别人的研究
+
写清自己的研究
```

很多本科生刚进入课题组时，会把“文献阅读”理解成英文翻译，把“科研写作”理解成把实验步骤写下来。这个理解不够。

在真实科研训练中，文献阅读和科研写作要服务于同一个目标：

```text
理解研究问题
→ 拆解实验设计
→ 读懂表征和性能结果
→ 建立工艺—组织—性能逻辑
→ 整理成可汇报、可复查、可积累的科研表达
```

因此，本模块不是单纯教你“怎么读文章”或“怎么写周报”，而是帮助你把文献、实验、数据和组会表达连接起来。

---

## 1. 本模块解决什么问题

本模块主要解决四类问题：

1. 如何阅读一篇材料论文；
2. 如何从论文中提取实验设计和组织—性能逻辑；
3. 如何规范记录自己的实验过程和原始数据；
4. 如何把每周科研训练整理成周报、组会汇报和保研科研经历。

如果你读论文时只是逐句翻译，写周报时只是写“这周做了 SEM，下周继续做”，说明还没有形成科研表达能力。

本模块的目标是让你逐步做到：

```text
论文读完能讲清主线；
实验做完能留下证据；
周报写完能说明进展；
组会汇报能体现思考。
```

## 3. 页面说明

### 3.1 如何阅读一篇材料论文

入口页面：

[如何阅读一篇材料论文](./paper-reading.md)

这篇文章主要解决：

```text
材料论文应该按什么顺序读？
如何抓住摘要、引言、实验方法、结果讨论和结论？
如何读 SEM、EDS、XRD、EBSD、TEM、拉伸和腐蚀图表？
如何从论文中提取工艺—组织—性能链条？
如何把文献内容整理成组会汇报和科研笔记？
```

适合在以下场景使用：

* 第一次读材料英文论文；
* 准备文献汇报；
* 想从论文中学习实验设计；
* 想积累论文写作表达；
* 准备保研面试中介绍文献阅读经历。

学习这篇文章时，不要把重点放在逐句翻译，而要重点训练：

```text
研究问题
→ 实验变量
→ 表征证据
→ 性能结果
→ 机制解释
→ 不足与启发
```

---

### 3.2 实验记录与周报模板

入口页面：

[实验记录与周报模板](./weekly-report.md)

这篇文章主要解决：

```text
实验记录应该记录哪些信息？
样品编号、工艺参数、测试条件和数据路径如何整理？
周报如何从流水账变成科研型表达？
如何写清楚本周目标、关键结果、初步分析、问题和下周计划？
如何把周报转化为组会汇报和保研科研经历？
```

适合在以下场景使用：

* 进入课题组开始做实验；
* 每周需要向学长或老师汇报；
* 实验数据和图片开始变多；
* 样品编号、路径和结果容易混乱；
* 想把科研训练积累成可复盘材料。

学习这篇文章时，要重点建立：

```text
可复查
可分析
可汇报
可积累
```

四个意识。

## 4. 文献阅读和科研写作的关系

文献阅读和科研写作不是两个孤立任务。

读论文时，你在学习别人如何提出问题、设计实验、组织图表、解释机制。

写周报时，你在训练自己如何记录实验、分析结果、说明问题、规划下一步。

二者之间的关系可以理解为：

```text
读论文
→ 学习别人的研究逻辑

写记录
→ 保存自己的实验证据

写周报
→ 整理自己的阶段进展

做组会
→ 训练自己的科研表达
```

如果只读论文，不写记录，知识很难转化为自己的项目能力。

如果只做实验，不读论文，很容易不知道结果应该如何解释。

## 5. 本模块的核心输出

学完本模块后，应逐步形成以下几类输出。

### 5.1 文献阅读记录

至少包括：

```text
论文题目
材料体系
研究问题
实验变量
表征方法
性能测试
关键结果
工艺—组织—性能链条
对自己课题的启发
```

文献笔记不是复制原文，而是把论文转化成自己的科研理解。

### 5.2 实验记录

至少包括：

```text
日期
样品编号
样品状态
实验目的
工艺参数
仪器条件
实验现象
原始数据路径
异常情况
初步判断
下一步计划
```

实验记录的底线是：以后能复查，别人能看懂。

### 5.3 科研周报

至少包括：

```text
本周目标
完成工作
关键结果
初步分析
遇到的问题
下周计划
数据路径
需要讨论的问题
```

周报不是“做了什么”的流水账，而是“围绕什么问题推进到了哪里”。

### 5.4 组会汇报素材

周报积累到一定程度后，可以转化为组会汇报：

```text
研究背景
→ 本周目标
→ 实验设计
→ 关键图表
→ 结果分析
→ 当前问题
→ 下一步计划
```

如果平时记录不规范，组会前就会非常痛苦。

## 6. 学习时最重要的判断标准

本模块学习时，要不断问自己：

```text
我是否知道这篇论文研究什么问题？
我是否能说清作者为什么这样设计实验？
我是否能解释一张图支撑了什么结论？
我是否能把自己的实验结果写成证据链？
我是否能说明当前结论还缺什么验证？
```

如果回答不了这些问题，说明你还停留在表面阅读和流水账记录阶段。

## 7. 常见误区

### 7.1 把文献阅读当成翻译

错误做法：

```text
逐句翻译摘要和引言
复制论文结论
不分析实验设计
不判断证据是否充分
```

正确做法：

```text
提取研究问题、实验变量、组织变化、性能响应和机制解释。
```

### 7.2 把周报写成流水账

错误做法：

```text
本周做了金相和 SEM。
下周继续做 XRD。
```

正确做法：

```text
本周围绕某个问题，完成了哪些实验，结果显示什么，可能说明什么，目前还有哪些不确定性，下周如何验证。
```

### 7.3 只保存图片，不保存原始数据

处理后的图不能替代原始数据。

每次实验后必须保存：

* 原始数据；
* 处理后图片；
* 样品编号；
* 测试条件；
* 数据路径；
* 异常情况。

### 7.4 结论写得太满

证据不足时不要写：

```text
完全证明
唯一原因
必然导致
彻底解决
```

更稳妥的表达是：

```text
可能与某机制有关
初步推测
仍需进一步验证
```

## 9. 学完本模块应达到什么程度

### 9.1 最低要求

至少应能做到：

1. 读完一篇论文后说清研究对象、方法和结论；
2. 能识别论文中的材料体系、实验变量和表征方法；
3. 每次实验后能记录样品编号、实验条件和数据路径；
4. 每周能写出一份不完全流水账的科研周报；
5. 能说明当前结果还缺哪些证据。

### 9.2 科研训练要求

进入课题组后，应进一步做到：

1. 能把论文整理成工艺—组织—性能链条；
2. 能把实验记录转化为组会汇报；
3. 能用图表和数据支撑自己的分析；
4. 能把问题写清楚，而不是只说“数据不对”；
5. 能提出下一步验证方案。

### 9.3 保研准备要求

如果希望把科研训练用于保研，应达到：

1. 能用文献阅读说明自己了解研究背景；
2. 能用周报积累证明自己持续参与科研；
3. 能从实验记录中提炼自己的贡献；
4. 能在面试中讲清楚项目逻辑和不足；
5. 能把科研经历写成简历、面试和个人陈述中的有效内容。

## 10. 推荐使用方式

如果你正在读论文：

```text
先看《如何阅读一篇材料论文》
再用文献阅读记录模板整理
最后提取对自己课题有用的实验设计和表达方式
```

如果你正在做实验：

```text
先看《实验记录与周报模板》
当天完成实验记录
每周整理一次周报
组会前从周报中提取关键结果
```

如果你准备组会或保研面试：

```text
先回顾周报
再梳理工艺—组织—性能链条
最后用面试问答页面检查表达是否清楚
```

## 11. 自检问题

学完本模块后，可以用下面问题检查自己：

1. 一篇材料论文应该先读哪些部分？
2. 如何从论文中提取研究问题？
3. 实验方法部分为什么要关注自变量、因变量和对照组？
4. 读 SEM、EDS、XRD 图时分别要注意什么？
5. 文献笔记为什么不能只是复制原文？
6. 实验记录为什么必须记录样品编号和数据路径？
7. 周报为什么不能写成流水账？
8. 初步分析中如何避免过度下结论？
9. 如何把周报转化为组会汇报？
10. 如何把文献阅读和实验记录转化为保研科研经历？
