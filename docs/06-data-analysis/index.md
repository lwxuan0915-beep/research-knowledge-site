# 数据处理与图谱分析

本模块用于训练金属材料科研中的基础数据分析能力。

科研训练不是只要“做出图”就可以。真正重要的是：

```text
数据从哪里来？
单位是否正确？
计算过程是否可复查？
图表是否能支撑结论？
结果是否和组织分析一致？
```

初学者常见问题是只保留最终图片，不保存原始数据；只看曲线高低，不理解曲线每一段含义；只比较单个数值，不考虑测试条件和重复性。

本模块的目标是帮助你建立数据可靠性意识。

## 2. 应力—应变曲线为什么重要

应力—应变曲线是金属材料科研中最常见的力学性能数据之一。

它能反映：

* 弹性变形；
* 屈服行为；
* 塑性变形；
* 加工硬化；
* 颈缩；
* 断裂；
* 强塑性匹配。

但应力—应变曲线不能孤立解释。

例如：

```text
强度提高
→ 可能来自晶粒细化、位错强化、固溶强化或析出强化
```

```text
延伸率下降
→ 可能与位错密度升高、粗大第二相、孔洞或裂纹有关
```

所以曲线分析最终要回到组织—性能关系。

## 4. 数据处理的基本原则

### 4.1 原始数据必须保存

处理后的图片不能替代原始数据。

必须保存：

* 原始载荷—位移数据；
* 原始应力—应变数据；
* 试样尺寸；
* 原始标距；
* 横截面积；
* 测试条件；
* 数据处理表格；
* 导出图表。

### 4.2 单位必须检查

常见单位包括：

* N；
* mm；
* MPa；
* GPa；
* %；
* mm/min；
* s⁻¹。

单位错误会直接导致结果错误。

### 4.3 测试条件必须记录

至少记录：

* 试样编号；
* 试样尺寸；
* 标距；
* 拉伸速率；
* 测试温度；
* 断裂位置；
* 是否为有效样品；
* 是否有重复样品。

### 4.4 图表必须支撑结论

图表不是装饰。每张图都应该能回答一个问题。

例如：

```text
这张曲线说明哪个样品强度更高？
这张柱状图是否显示误差？
这个结果是否和组织变化一致？
```

## 5. 常见数据分析误区

### 5.1 只看抗拉强度

抗拉强度高不代表综合性能一定好。

还要看：

* 屈服强度；
* 延伸率；
* 加工硬化能力；
* 断裂方式；
* 断口形貌；
* 服役环境。

### 5.2 不区分工程应力和真实应力

工程应力使用原始横截面积，真实应力使用瞬时横截面积。

颈缩后，工程应力下降并不等于局部真实应力一定下降。

### 5.3 曲线异常不排查

初始段异常弯曲、突然跳变、锯齿波动、断裂位置异常，都可能来自测试或试样问题。

不能全部归因于材料本身。

### 5.4 没有重复样品

单个样品不能代表材料整体性能。科研训练中应尽量保留重复样品和误差信息。

## 6. 数据分析与组织—性能关系

数据分析不能只停留在数字层面。

例如：

```text
屈服强度提高
```

应进一步追问：

```text
是晶粒细化导致？
是位错密度增加导致？
是析出相强化导致？
还是测试条件不同导致？
```

再如：

```text
延伸率下降
```

应进一步追问：

```text
是否有粗大第二相？
是否存在孔洞或裂纹？
断口是否出现脆性特征？
组织是否发生严重加工硬化？
```

力学数据必须和显微组织、断口和工艺条件共同解释。

## 8. 学完本模块应达到什么程度

### 8.1 最低要求

至少应能做到：

1. 看懂应力—应变曲线的基本阶段；
2. 知道屈服强度、抗拉强度、延伸率的含义；
3. 明白强度和塑性不能只看一个指标；
4. 能识别明显异常曲线；
5. 能保存原始数据和处理过程。

### 8.2 科研训练要求

进入课题组后，应进一步做到：

1. 能提取主要力学性能指标；
2. 能比较不同样品的强塑性变化；
3. 能结合组织结果解释曲线变化；
4. 能判断测试异常；
5. 能用规范语言写出数据分析段落。

## 9. 自检问题

1. 当前曲线是工程应力—应变还是真实应力—应变？
2. 试样尺寸和标距是否记录？
3. 屈服强度如何提取？
4. 抗拉强度是否为最大工程应力？
5. 延伸率是否来自有效断裂样品？
6. 是否有重复样品？
7. 曲线初始段是否异常？
8. 强度变化是否有组织证据支撑？
9. 是否只看强度而忽略塑性？
10. 原始数据是否保存？
