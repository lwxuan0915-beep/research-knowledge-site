# 应力—应变曲线怎么读

应力—应变曲线是金属材料科研训练中最基础、最常见的力学性能数据之一。

很多本科生第一次看到拉伸曲线时，只会说：

> 这个样品强度高。
> 这个样品塑性好。
> 这个曲线断了。

这些说法太粗糙。真正的材料科研分析需要进一步回答：

> 曲线的每一段分别对应什么变形阶段？
> 屈服强度、抗拉强度、延伸率分别说明什么？
> 为什么强度提高后塑性可能下降？
> 曲线变化能否用晶粒、位错、第二相、析出相或缺陷解释？
> 这个性能结果是否和组织表征结果一致？

应力—应变曲线不是孤立数据，而是连接材料组织和宏观性能的重要证据。

---

## 1. 本文学习目标

学完本文后，应能做到：

1. 区分应力、应变、工程应力、工程应变、真实应力、真实应变；
2. 看懂拉伸曲线中的弹性阶段、屈服阶段、塑性阶段、颈缩阶段和断裂阶段；
3. 理解弹性模量、屈服强度、抗拉强度、断后伸长率、断面收缩率的含义；
4. 知道 0.2% 偏移屈服强度为什么常用于无明显屈服平台的金属；
5. 能从曲线形状判断材料强度、塑性、加工硬化和断裂倾向；
6. 能把应力—应变曲线与显微组织变化联系起来；
7. 能避免只看强度、不看塑性和测试条件的常见误区。

---

## 2. 为什么要学习应力—应变曲线

金属材料科研经常需要回答：

```text
某种工艺是否改善了材料性能？
某种组织变化是否提高了强度？
材料是否在提高强度的同时牺牲了塑性？
断裂行为是否发生改变？
```

应力—应变曲线可以提供最直接的力学性能证据。

例如：

```text
热处理
→ 析出相变化
→ 位错运动受阻程度变化
→ 屈服强度和抗拉强度变化
```

又如：

```text
冷变形
→ 位错密度增加
→ 加工硬化增强
→ 强度提高但延伸率下降
```

再如：

```text
晶粒细化
→ 晶界数量增加
→ 位错运动受阻
→ 屈服强度提高
```

因此，应力—应变曲线不是单纯的数据图，而是材料组织—性能关系中的关键证据。

---

## 3. 什么是应力

应力表示单位面积上承受的力。

工程拉伸测试中常用工程应力：

```text
工程应力 = 拉伸载荷 / 原始横截面积
```

通常记为：

```text
σ = F / A0
```

其中：

* σ 表示工程应力；
* F 表示拉伸过程中施加的载荷；
* A0 表示试样原始横截面积。

常见单位包括：

```text
MPa
GPa
```

在金属材料论文中，屈服强度和抗拉强度通常以 MPa 表示。

---

## 4. 什么是应变

应变表示材料长度变化相对于原始长度的比例。

工程应变定义为：

```text
工程应变 = 长度变化量 / 原始标距长度
```

通常记为：

```text
ε = ΔL / L0
```

其中：

* ε 表示工程应变；
* ΔL 表示长度变化量；
* L0 表示原始标距长度。

应变本身没有单位，常用百分数表示，例如：

```text
10%
20%
30%
```

---

## 5. 工程应力—应变与真实应力—应变

### 5.1 工程应力—应变

工程应力和工程应变都基于试样的原始尺寸计算。

工程应力使用原始横截面积，工程应变使用原始标距长度。

它的优点是：

* 计算简单；
* 工程应用广；
* 论文中常见；
* 便于比较不同样品。

但它的缺点是：

* 没有考虑拉伸过程中截面积不断减小；
* 颈缩之后不能真实反映局部变形状态。

### 5.2 真实应力—应变

真实应力考虑瞬时横截面积，真实应变考虑连续变形过程。

在均匀塑性变形阶段，常见近似关系为：

```text
真实应力 = 工程应力 × (1 + 工程应变)
真实应变 = ln(1 + 工程应变)
```

需要注意：

> 这些简单换算通常适用于颈缩前的均匀变形阶段。颈缩后局部变形复杂，不能机械套用简单公式。

本科科研训练初期，通常先掌握工程应力—应变曲线，再逐步理解真实应力—应变曲线。

---

## 6. 拉伸曲线的基本阶段

典型金属工程应力—应变曲线通常可以分为几个阶段：

```text
弹性变形阶段
→ 屈服阶段
→ 均匀塑性变形阶段
→ 颈缩阶段
→ 断裂阶段
```

不同材料的曲线形状会有差异。例如，低碳钢可能有明显屈服平台，而铝合金、不锈钢、钛合金等材料可能没有明显屈服点。

---

## 7. 弹性变形阶段

弹性变形是指卸载后可以恢复的变形。

在曲线初始阶段，应力和应变近似呈线性关系：

```text
σ = Eε
```

其中 E 是弹性模量，也叫杨氏模量。

### 7.1 弹性模量的含义

弹性模量表示材料抵抗弹性变形的能力。

弹性模量越大，在相同应力下弹性应变越小。

例如，钢的弹性模量通常明显高于铝合金，因此在相同应力水平下，钢的弹性变形通常更小。

### 7.2 弹性阶段的科研意义

弹性阶段可以反映材料刚度，但要注意：

> 弹性模量主要由原子键合和晶体结构等因素决定，普通热处理或少量组织变化通常不会像强度那样显著改变弹性模量。

如果实验中不同样品弹性模量差异特别大，应优先怀疑测试、夹持、引伸计、试样尺寸或数据处理问题，而不是直接归因于组织变化。

---

## 8. 屈服阶段

屈服表示材料开始发生明显塑性变形。

屈服之前，材料主要发生弹性变形；屈服之后，卸载后会保留不可恢复的塑性变形。

常见屈服指标包括：

* 上屈服强度；
* 下屈服强度；
* 0.2% 偏移屈服强度。

### 8.1 明显屈服

某些材料会出现明显屈服平台，例如部分低碳钢。

曲线上可能出现：

```text
上屈服点
→ 应力下降
→ 屈服平台
→ 后续加工硬化
```

这种现象与位错运动、间隙原子钉扎等机制有关。

### 8.2 无明显屈服

很多金属材料没有清晰屈服平台，例如铝合金、铜合金、奥氏体不锈钢等。

这时通常采用 0.2% 偏移屈服强度。

### 8.3 0.2% 偏移屈服强度

0.2% 偏移屈服强度是人为定义的工程指标。

基本做法是：

```text
在应变轴上取 0.2% 塑性应变偏移
→ 作一条与弹性阶段斜率平行的直线
→ 该直线与应力—应变曲线的交点对应的应力
→ 记为 0.2% 偏移屈服强度
```

它的作用是为无明显屈服平台的材料提供可比较的屈服强度指标。

---

## 9. 塑性变形阶段

屈服后，材料进入塑性变形阶段。

塑性变形的本质之一是位错运动。

基本逻辑为：

```text
外加载荷增加
→ 晶体中剪应力增加
→ 位错在滑移面上运动
→ 原子逐步错排
→ 材料产生不可恢复变形
```

塑性阶段对金属材料非常重要，因为工程材料不仅要有强度，还要有足够变形能力，避免突然脆断。

---

## 10. 加工硬化阶段

在塑性变形过程中，应力继续升高，这通常与加工硬化有关。

加工硬化的基本逻辑是：

```text
塑性变形增加
→ 位错密度升高
→ 位错之间相互缠结和阻碍
→ 后续位错运动更困难
→ 需要更高应力继续变形
```

因此，应力—应变曲线在塑性阶段继续上升。

### 10.1 加工硬化的意义

加工硬化能力强的材料，通常具有较好的均匀塑性变形能力。

如果材料几乎没有加工硬化能力，可能更容易发生局部变形和早期颈缩。

### 10.2 科研训练中的表达

可以这样表达：

```text
该样品在屈服后仍表现出明显应力上升，说明其具有一定加工硬化能力。结合显微组织分析，这可能与位错增殖、第二相阻碍位错运动或多滑移系协调变形有关。
```

不要只写：

```text
曲线升高，说明性能变好。
```

这类表达太粗糙。

---

## 11. 抗拉强度

抗拉强度是工程应力—应变曲线上的最大工程应力。

通常记为：

```text
UTS
```

或：

```text
Rm
```

抗拉强度表示材料在拉伸过程中承受最大工程载荷的能力。

需要注意：

> 抗拉强度不是断裂强度。工程应力—应变曲线中，最大应力通常出现在颈缩开始附近，而断裂点可能对应较低工程应力。

### 11.1 抗拉强度的科研意义

抗拉强度常用于比较不同工艺样品的整体强度水平。

但不能只看抗拉强度，还要同时看：

* 屈服强度；
* 延伸率；
* 加工硬化行为；
* 断口形貌；
* 数据离散性；
* 强塑性匹配。

---

## 12. 颈缩阶段

颈缩是指拉伸过程中试样局部区域发生明显截面积减小。

在工程应力—应变曲线上，颈缩后工程应力通常下降。

这不是因为材料局部真实承载能力一定下降，而是因为工程应力仍然用原始横截面积计算，不能反映局部真实截面积急剧减小的影响。

### 12.1 颈缩的意义

颈缩说明材料变形开始局部化。

颈缩发生后，试样不再均匀变形，局部应变集中，最终导致断裂。

### 12.2 颈缩与塑性

延伸率高的材料通常能承受较大塑性变形，但仅凭是否颈缩不能完整判断塑性。

还需要结合：

* 断后伸长率；
* 断面收缩率；
* 断口形貌；
* 均匀延伸率；
* 局部延伸率。

---

## 13. 断裂阶段

断裂是拉伸试样最终失效的阶段。

断裂行为通常分为：

* 韧性断裂；
* 脆性断裂；
* 混合断裂；
* 沿晶断裂；
* 穿晶断裂。

### 13.1 韧性断裂

韧性断裂通常伴随明显塑性变形。

SEM 断口上常见：

* 韧窝；
* 微孔聚合；
* 撕裂棱；
* 颈缩特征。

### 13.2 脆性断裂

脆性断裂通常塑性变形较少。

断口上可能出现：

* 解理台阶；
* 河流花样；
* 沿晶断裂形貌；
* 平直断裂面。

### 13.3 曲线与断口要结合

应力—应变曲线只能告诉你宏观力学响应，断裂机制还需要结合断口分析。

例如：

```text
曲线显示延伸率明显降低
→ SEM 断口出现解理面或沿晶裂纹
→ 说明材料断裂倾向可能由韧性向脆性转变
```

---

## 14. 延伸率

延伸率表示试样断裂后长度增加的比例。

常见表达为：

```text
断后伸长率 = (断后标距长度 - 原始标距长度) / 原始标距长度 × 100%
```

延伸率常用于衡量材料塑性。

但要注意：

> 延伸率与试样标距、尺寸、测试标准和断裂位置有关，不同实验条件下不能随意横向比较。

### 14.1 均匀延伸率和总延伸率

总延伸率包括：

```text
均匀塑性变形
+
颈缩后的局部塑性变形
```

均匀延伸率更能反映材料在颈缩前稳定塑性变形的能力。

对于科研分析，如果条件允许，应区分均匀延伸率和断后总延伸率。

---

## 15. 断面收缩率

断面收缩率表示断裂后截面积减小的比例。

它也是衡量塑性的重要指标。

一般来说，断面收缩率越大，说明材料局部塑性变形能力越强。

但它同样受试样形状、断裂位置和测试条件影响。

---

## 16. 强度与塑性的关系

材料科研中经常出现强度和塑性之间的矛盾。

常见情况是：

```text
强度提高
→ 塑性下降
```

例如冷变形后：

```text
位错密度增加
→ 位错运动受阻
→ 强度提高
→ 继续塑性变形能力下降
→ 延伸率降低
```

但强度和塑性并不永远只能二选一。通过合理组织设计，有可能实现较好的强塑性匹配。

例如：

* 细晶强化；
* 析出强化；
* 多尺度组织；
* 异质结构；
* 相变诱导塑性；
* 孪生诱导塑性。

本科阶段不需要马上掌握所有高级机制，但必须知道：

> 好材料不只是强度高，还要看强度、塑性、韧性、稳定性和服役环境之间的综合平衡。

---

## 17. 曲线形状如何反映材料行为

### 17.1 曲线很陡

初始弹性阶段曲线很陡，说明弹性模量较高。

但如果曲线初始段异常弯曲，可能是：

* 夹具滑移；
* 试样未拉直；
* 引伸计问题；
* 数据采集误差；
* 机器间隙未消除。

不能直接把异常初始段解释为材料本身性能。

### 17.2 屈服后迅速断裂

如果材料刚屈服不久就断裂，说明塑性储备较差。

可能原因包括：

* 脆性第二相；
* 裂纹或孔洞；
* 晶界脆化；
* 热处理不当；
* 氢脆；
* 严重应力集中；
* 试样加工缺陷。

### 17.3 屈服后曲线持续上升

这通常说明材料有一定加工硬化能力。

可能与位错增殖、析出相强化、多相组织协调变形等因素有关。

### 17.4 强度升高但延伸率降低

这是金属材料中很常见的现象。

不能只写“性能提高”，而要说明：

```text
强度提高说明变形抗力增加；
延伸率降低说明塑性变形能力下降；
该工艺提高强度的同时牺牲了部分塑性。
```

### 17.5 曲线波动明显

曲线波动可能来自：

* 材料不均匀变形；
* 锯齿流变；
* 动态应变时效；
* 夹具或测试系统问题；
* 数据采集噪声；
* 试样局部缺陷。

需要结合材料体系和测试条件判断，不能一概而论。

---

## 18. 应力—应变曲线与显微组织的联系

应力—应变曲线必须回到组织中解释。

### 18.1 晶粒尺寸

晶粒细化通常会提高屈服强度。

逻辑是：

```text
晶粒细化
→ 晶界数量增加
→ 位错运动受阻
→ 屈服强度提高
```

### 18.2 位错密度

位错密度增加通常会提高强度，但可能降低塑性。

逻辑是：

```text
位错密度增加
→ 位错缠结增强
→ 后续位错运动困难
→ 强度提高
→ 延伸率下降
```

### 18.3 析出相

细小弥散析出相可以阻碍位错运动，提高强度。

但粗大析出相可能成为裂纹源，降低塑性和韧性。

### 18.4 第二相

第二相对性能的影响取决于：

* 尺寸；
* 形貌；
* 分布；
* 界面结合；
* 是否连续分布；
* 是否脆性；
* 是否诱发应力集中。

不能简单说“第二相提高强度”或“第二相降低性能”。

### 18.5 孔洞和裂纹

孔洞、夹杂物和微裂纹通常会降低塑性和疲劳性能。

如果曲线显示早期断裂，应考虑是否存在这些缺陷。

---

## 19. 拉伸数据处理时要记录什么

处理拉伸数据时，至少应记录：

1. 材料名称和状态；
2. 试样编号；
3. 试样尺寸；
4. 原始标距；
5. 横截面积；
6. 拉伸速度或应变速率；
7. 测试温度；
8. 屈服强度；
9. 抗拉强度；
10. 断后伸长率；
11. 断面收缩率；
12. 是否有重复样品；
13. 平均值和标准差；
14. 断裂位置是否有效；
15. 曲线是否异常。

科研训练中不能只保留最终曲线图片，原始数据和处理过程也必须保存。

## 20. 论文中如何表达拉伸结果

低质量表达：

```text
样品 A 的强度比样品 B 高，所以样品 A 更好。
```

问题是只看强度，没有看塑性和机制。

更好的表达：

```text
与样品 B 相比，样品 A 的屈服强度和抗拉强度均有所提高，说明其变形抗力增强。然而，样品 A 的断后伸长率降低，表明该处理在提高强度的同时牺牲了部分塑性。结合显微组织结果，强度提高可能与晶粒细化和位错密度增加有关，而塑性下降可能与位错缠结和局部应力集中有关。
```

再如：

```text
该样品屈服后仍表现出明显加工硬化阶段，说明其在塑性变形过程中仍能持续积累位错并阻碍后续滑移。这种加工硬化能力有助于延缓颈缩，提高均匀塑性变形能力。
```

## 21. 常见误区

### 误区 1：只看抗拉强度

抗拉强度高不代表材料一定更好。

还要看：

* 屈服强度；
* 延伸率；
* 韧性；
* 疲劳性能；
* 腐蚀性能；
* 断裂方式；
* 服役环境。

### 误区 2：把强度提高直接等同于性能改善

如果强度提高但塑性大幅下降，材料实际应用价值可能并没有提高。

要写成：

```text
强度提高，但强塑性匹配需要进一步评价。
```

### 误区 3：不区分工程应力和真实应力

工程应力使用原始面积，真实应力使用瞬时面积。

颈缩后，工程应力下降并不等于局部真实应力一定下降。

### 误区 4：随意比较不同测试条件的数据

不同试样尺寸、标距、拉伸速度、测试温度、热处理状态下的数据不能简单横向比较。

### 误区 5：把曲线异常全部归因于材料

曲线初始段异常、突跳、锯齿、突然下降，可能来自测试系统或试样问题。

要先排查：

* 夹持是否滑移；
* 试样尺寸是否准确；
* 断裂位置是否在标距内；
* 引伸计是否正常；
* 数据采集是否稳定。

### 误区 6：没有重复样品

单个试样的曲线不能代表材料整体性能。

科研训练中应尽量有重复样品，并报告平均值和误差。

### 误区 7：只报数据，不解释组织原因

拉伸测试结果必须和组织表征结合。

例如：

```text
强度提高
→ 可能由晶粒细化、位错密度增加、固溶强化、析出强化共同导致
```

不能只写“拉伸结果显示强度提高”。

## 22. 本科生应掌握到什么程度

### 22.1 最低掌握要求

至少应能做到：

1. 看懂应力—应变曲线的基本形状；
2. 知道弹性阶段、屈服阶段、塑性阶段、颈缩阶段和断裂阶段；
3. 会解释屈服强度、抗拉强度和延伸率；
4. 知道强度和塑性不能只看一个指标；
5. 能识别明显异常曲线。

### 22.2 科研训练要求

进入课题组后，应进一步做到：

1. 能从曲线中提取主要力学性能指标；
2. 能比较不同样品的强塑性变化；
3. 能结合组织表征解释曲线变化；
4. 能判断曲线异常可能来自材料还是测试问题；
5. 能用规范语言描述拉伸结果。

### 22.3 保研科研经历要求

如果希望把科研训练用于保研面试，应达到：

1. 能用一张应力—应变曲线说明材料的变形过程；
2. 能解释某种工艺为什么提高强度或降低塑性；
3. 能把拉伸数据和 SEM、EBSD、TEM、XRD 结果联系起来；
4. 能说明自己如何处理数据和判断结果可靠性；
5. 能指出结果中的不足，例如重复样品不足或缺少断口分析。

## 23. 组会汇报模板

可以使用下面模板汇报拉伸结果：

```text
本实验对 ______ 状态样品进行了拉伸测试。
从应力—应变曲线可以看出，______ 样品的屈服强度和抗拉强度分别表现为 ______。
与对照样品相比，其强度变化说明 ______。
同时，延伸率变化表明 ______。
结合显微组织结果，强度变化可能与 ______ 有关，而塑性变化可能与 ______ 有关。
后续还需要结合断口形貌或更多重复样品进一步验证。
```

示例：

```text
本实验对不同时效状态的铝合金样品进行了拉伸测试。从应力—应变曲线可以看出，峰值时效样品的屈服强度和抗拉强度明显高于未时效样品，说明其变形抗力提高。同时，峰值时效样品延伸率有所下降，说明塑性受到一定影响。结合组织分析，强度提高可能与细小弥散析出相阻碍位错运动有关，而塑性下降可能与析出相引起的局部应力集中有关。后续还需要结合 TEM 析出相观察和 SEM 断口分析进一步验证。
```

## 24. 数据处理检查清单

处理应力—应变曲线时，建议逐项检查：

* [ ] 试样编号是否清楚；
* [ ] 原始尺寸是否记录；
* [ ] 标距是否记录；
* [ ] 载荷单位是否正确；
* [ ] 位移或应变单位是否正确；
* [ ] 横截面积是否计算正确；
* [ ] 工程应力是否使用原始面积；
* [ ] 工程应变是否使用原始标距；
* [ ] 屈服强度提取方法是否说明；
* [ ] 是否需要 0.2% 偏移屈服强度；
* [ ] 抗拉强度是否为最大工程应力；
* [ ] 延伸率是否来自断后测量或曲线计算；
* [ ] 是否有重复样品；
* [ ] 是否报告平均值和误差；
* [ ] 曲线异常是否被排查；
* [ ] 断裂位置是否有效；
* [ ] 是否结合组织结果解释性能变化。
