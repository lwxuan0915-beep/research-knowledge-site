# 实验室基础技能

本模块面向刚进入金属材料课题组的本科生，重点解决一个问题：

```text
进入实验室以后，最基础、最容易出错、最需要规范的操作是什么？
```

对于金属材料科研训练来说，实验室技能不是简单的“会不会操作仪器”，而是包括：

```text
样品准备
→ 样品编号
→ 制样质量
→ 实验安全
→ 原始记录
→ 后续表征可靠性
```

很多后续 SEM、EDS、XRD、金相观察和力学测试中的问题，根源并不在仪器本身，而在前期样品准备和记录不规范。

---

## 1. 本模块解决什么问题

本模块主要帮助你建立以下意识：

1. 样品制备质量会直接影响后续表征结果；
2. 金相制样不是“磨亮”这么简单，而是为了真实显示组织；
3. 样品编号和实验记录是后续数据可追溯的基础；
4. 实验安全是科研训练的底线；
5. 任何实验操作都要服务于后续组织—性能分析。

## 3. 金相制样在科研训练中的作用

金相制样是很多金属材料实验的第一步。

它通常包括：

```text
切割
→ 镶嵌
→ 研磨
→ 抛光
→ 腐蚀
→ 显微观察
```

每一步都会影响最终观察结果。

例如：

* 切割过热可能改变表面组织；
* 研磨不充分会留下划痕；
* 抛光不好会影响 SEM 或金相图像；
* 腐蚀过度可能造成晶界和第二相失真；
* 样品编号混乱会导致图像和实验条件无法对应。

因此，金相制样的核心目标不是“把样品做得好看”，而是：

```text
获得能够真实反映材料显微组织的观察表面。
```

## 4. 本模块与其他模块的关系

### 4.1 与材料表征模块的关系

SEM、EDS、XRD 等表征结果的可靠性，往往依赖于样品前处理。

如果样品表面有划痕、污染、腐蚀过度或编号错误，后续表征结果就可能被误读。

建议配合学习：

* [SEM 扫描电镜入门](../03-characterization/sem.md)
* [EDS 能谱分析入门](../03-characterization/eds.md)
* [XRD 物相分析入门](../03-characterization/xrd.md)

### 4.2 与实验记录模块的关系

每次制样都应记录：

* 样品编号；
* 样品状态；
* 砂纸目数；
* 抛光步骤；
* 腐蚀剂；
* 腐蚀时间；
* 异常情况；
* 图片路径。

建议配合学习：

[实验记录与周报模板](../05-literature-writing/weekly-report.md)

### 4.3 与组织—性能分析模块的关系

金相观察不是最终目的。最终要把观察到的组织特征用于解释性能变化。

建议配合学习：

[工艺—组织—性能逻辑链进阶版](../04-microstructure-property/process-structure-property.md)

## 5. 学完本模块应达到什么程度

### 5.1 最低要求

至少应能做到：

1. 知道金相制样的基本流程；
2. 知道研磨、抛光和腐蚀分别有什么作用；
3. 明白划痕、过腐蚀、欠腐蚀会影响组织判断；
4. 能记录样品编号和制样条件；
5. 有基本实验安全意识。

### 5.2 科研训练要求

进入课题组后，应进一步做到：

1. 能按要求完成基础样品制备；
2. 能判断金相样品是否适合观察；
3. 能区分制样痕迹和真实组织特征；
4. 能把制样条件写入实验记录；
5. 能说明制样质量如何影响后续表征。

## 6. 常见误区

### 误区 1：认为样品磨亮就可以

样品表面亮不等于组织显示正确。关键是表面是否平整、划痕是否去除、腐蚀是否合适、组织是否真实。

### 误区 2：不记录腐蚀时间

腐蚀时间会直接影响晶界和组织显示。没有记录，后续无法复现。

### 误区 3：把划痕当成组织

研磨或抛光划痕可能被误认为裂纹、晶界或腐蚀痕迹，需要谨慎区分。

### 误区 4：样品编号混乱

样品编号混乱会导致后续 SEM、EDS、XRD、硬度和拉伸结果无法对应。
