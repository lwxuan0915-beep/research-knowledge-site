# 金相制样基础流程

## 1. 实验目的

金相制样的核心目的，是把金属材料内部或表面的组织状态，以尽可能真实、清晰、可观察的方式暴露出来，为后续显微组织分析提供可靠样品。

金相制样不是简单地“把样品磨亮”。真正合格的金相样品应满足以下要求：

1. 观察面平整；
2. 表面划痕少；
3. 无明显变形层；
4. 无严重拖尾、污染、脱落或边缘塌陷；
5. 腐蚀后组织清晰；
6. 能真实反映材料的晶粒、相、夹杂、第二相、孔洞、裂纹等特征。

如果制样质量不好，后续即使使用再好的显微镜，也可能得到错误判断。例如：

* 抛光划痕可能被误认为裂纹；
* 过腐蚀可能掩盖真实晶界；
* 制样变形层可能改变表面组织；
* 第二相脱落后留下的孔洞可能被误判为材料本身缺陷；
* 腐蚀不充分会导致晶粒、相界和组织形貌看不清。

所以金相制样是材料实验中非常基础但非常关键的一步。

---

## 2. 适用场景

金相制样常用于以下研究或实验场景：

1. 观察金属材料显微组织；
2. 分析晶粒尺寸；
3. 判断热处理后的组织变化；
4. 观察第二相、夹杂物或析出物分布；
5. 分析焊接接头的焊缝区、热影响区和母材组织；
6. 观察裂纹、孔洞、腐蚀坑等缺陷；
7. 为 OM、SEM、EDS、EBSD 等表征方法准备样品；
8. 为组织—性能关系分析提供证据。

在金属材料科研训练中，金相制样通常是连接“实验样品”和“组织分析”的第一道关口。

---

## 3. 金相制样的基本逻辑

金相制样通常遵循以下流程：

```text
取样 / 切割
→ 镶嵌
→ 粗磨
→ 细磨
→ 抛光
→ 清洗干燥
→ 腐蚀
→ 显微观察
→ 图片记录与分析
```

每一步都不是孤立的。前一步如果做得不好，后一步很难完全补救。

例如：

* 切割时过热，会引入热影响层；
* 磨样时用力过大，会形成表面变形层；
* 抛光不充分，会保留划痕；
* 腐蚀过度，会破坏真实组织；
* 清洗不干净，会造成污染和假象。

因此，金相制样的核心不是“最后抛亮”，而是全过程控制。

## 4. 所需设备与材料

常见设备包括：

1. 切割机；
2. 镶嵌机或冷镶嵌模具；
3. 砂纸；
4. 磨抛机；
5. 抛光布；
6. 抛光液或抛光膏；
7. 腐蚀剂；
8. 酒精；
9. 蒸馏水或去离子水；
10. 吹风装置或冷风吹干设备；
11. 光学显微镜；
12. 记录本或电子实验记录表。

常见耗材包括：

1. 不同目数砂纸；
2. 抛光布；
3. 金刚石抛光液或氧化铝抛光液；
4. 镶嵌粉或冷镶嵌树脂；
5. 棉签；
6. 滤纸；
7. 手套；
8. 镊子。

注意：不同材料体系对应的腐蚀剂不同。不能随意套用腐蚀剂，必须结合材料类型、实验目的和实验室规范选择。

## 5. 标准操作流程

## 5.1 取样与切割

### 5.1.1 目的

切割的目的是从原始材料中获得适合观察和后续制样的小样品。

切割时需要考虑：

1. 观察面的位置；
2. 材料的方向性；
3. 是否需要观察横截面或纵截面；
4. 是否需要保留裂纹、焊缝、腐蚀区或缺陷区；
5. 是否避免破坏关键区域。

例如，研究焊接接头时，切割面通常要覆盖：

```text
母材 → 热影响区 → 焊缝区 → 热影响区 → 母材
```

研究腐蚀坑时，切割方向需要保留腐蚀坑截面，避免把关键区域切掉。

### 5.1.2 关键控制点

切割时应注意：

1. 避免切割过热；
2. 使用合适的切割速度；
3. 必要时使用冷却液；
4. 不要让样品剧烈振动；
5. 不要切掉待观察区域；
6. 切割后标记样品方向。

切割过热可能造成局部组织变化，尤其对热处理材料、焊接材料、变形材料影响较大。切割产生的热影响层会干扰后续组织观察。

---

## 5.2 镶嵌

### 5.2.1 目的

镶嵌的目的是方便拿持、保护边缘、保证磨抛时样品稳定。

小样品、薄片样品、形状不规则样品通常需要镶嵌。

### 5.2.2 热镶嵌与冷镶嵌

常见镶嵌方式包括：

1. 热镶嵌；
2. 冷镶嵌。

热镶嵌速度较快，样品固定效果较好，但不适合对温度敏感的样品。

冷镶嵌适合形状复杂、易受热影响或需要保留特殊表面的样品，但固化时间较长。

### 5.2.3 常见问题

镶嵌常见问题包括：

1. 样品倾斜；
2. 样品边缘未被保护；
3. 树脂与样品之间有缝隙；
4. 样品位置不利于后续观察；
5. 样品方向标记丢失。

如果镶嵌时样品倾斜，后续磨抛可能导致观察面不完整。
如果样品边缘保护不好，后续容易出现边缘塌陷，影响边缘组织或表面层观察。

---

## 5.3 磨样

## 5.3.1 目的

磨样的目的是去除切割损伤层和粗糙表面，使样品表面逐步变平整。

磨样通常从粗砂纸到细砂纸逐级进行。

一般逻辑是：

```text
粗磨：快速去除切割痕迹和不平整表面
细磨：逐步减小划痕深度
最终磨样：为抛光创造条件
```

## 5.3.2 砂纸目数变化

常见做法是从低目数到高目数逐级磨样，例如：

```text
粗砂纸 → 中等砂纸 → 细砂纸 → 更细砂纸
```

具体目数应根据实验室条件和材料硬度确定。原则是：不能跳级太大，否则前一级粗划痕难以被后一级去除。

## 5.3.3 磨样方向控制

每换一级砂纸，建议将样品旋转约 90°，使新磨痕方向与上一道磨痕方向垂直。

这样做的原因是：

1. 方便判断上一道划痕是否完全去除；
2. 避免粗划痕残留；
3. 提高后续抛光效率。

当显微观察或肉眼观察发现上一道方向的划痕基本消失后，才能进入下一道砂纸。

## 5.3.4 磨样用力

磨样不能一味用力。用力过大可能导致：

1. 表面塑性变形；
2. 局部过热；
3. 软相拖拽；
4. 第二相脱落；
5. 边缘塌陷；
6. 样品表面不平。

正确做法是保持均匀压力，让样品表面稳定接触砂纸。

---

## 5.4 抛光

## 5.4.1 目的

抛光的目的是去除细磨后残留的细小划痕，使表面达到适合显微观察的镜面状态。

合格抛光表面通常应满足：

1. 表面明亮；
2. 划痕很少；
3. 无明显拖尾；
4. 无明显污染；
5. 无明显凹坑或第二相脱落；
6. 清洗后无残留抛光液。

## 5.4.2 抛光过程

抛光时需要控制：

1. 抛光布类型；
2. 抛光液种类；
3. 抛光时间；
4. 抛光压力；
5. 抛光盘转速；
6. 样品移动方式；
7. 清洗是否及时。

对于含有软硬相差异明显的材料，抛光时尤其要注意拖尾和选择性磨损。

## 5.4.3 常见抛光问题

### 1. 划痕残留

原因可能是：

1. 前一道砂纸划痕未完全去除；
2. 抛光时间不足；
3. 抛光布污染；
4. 抛光液颗粒不合适。

### 2. 拖尾

拖尾常见于软相、夹杂物或第二相附近。其表现为组织边缘被拉长，导致形貌失真。

### 3. 第二相脱落

对于含有脆性第二相、夹杂物或孔洞的样品，抛光过程中可能出现局部脱落，留下黑色坑洞。

这类坑洞不能直接判断为材料原始孔洞，必须结合位置、形貌和其他表征方法判断。

### 4. 边缘塌陷

边缘塌陷会导致样品边缘区域组织失真，特别影响表面处理层、腐蚀层、镀层或薄壁样品的观察。

---

## 5.5 清洗与干燥

抛光后必须及时清洗样品，避免抛光液、腐蚀剂或污染物残留。

常见步骤：

```text
水洗
→ 酒精清洗
→ 吹干
→ 立即观察或保存
```

清洗不彻底可能导致：

1. 表面污染；
2. 腐蚀不均匀；
3. 显微镜下出现假斑点；
4. 后续 EDS 成分分析受到干扰。

样品清洗后应尽快吹干，避免水迹残留或二次腐蚀。

---

## 5.6 腐蚀

## 5.6.1 腐蚀的目的

腐蚀的目的是通过不同组织、晶粒、相或晶界之间的化学反应差异，使显微组织显现出来。

抛光后的金属表面虽然光亮，但很多组织细节无法直接观察。腐蚀后，晶界、相界、第二相或不同取向晶粒之间会产生衬度差异。

## 5.6.2 腐蚀剂选择

腐蚀剂必须根据材料体系选择。

不同材料的腐蚀剂不同，例如钢、铝合金、铜合金、钛合金、镍基合金通常不能混用同一种腐蚀方法。

使用腐蚀剂前必须确认：

1. 材料类型；
2. 实验目的；
3. 腐蚀剂配方；
4. 腐蚀时间；
5. 安全要求；
6. 废液处理方式。

## 5.6.3 腐蚀时间控制

腐蚀时间过短，组织显示不清楚，称为欠腐蚀。

腐蚀时间过长，组织边界可能变粗、变黑甚至被破坏，称为过腐蚀。

判断腐蚀是否合适，不能只看肉眼颜色，要结合显微镜观察。

## 5.6.4 常见腐蚀问题

### 欠腐蚀

表现为：

1. 晶界不明显；
2. 组织衬度低；
3. 第二相难以分辨；
4. 图片整体发白或缺少层次。

### 过腐蚀

表现为：

1. 晶界过粗；
2. 组织发黑；
3. 局部腐蚀坑严重；
4. 表面真实形貌被破坏；
5. 难以区分真实缺陷和腐蚀造成的假象。

---

## 6. 结果判断

一个基本合格的金相样品应满足：

1. 观察面平整；
2. 无明显粗大划痕；
3. 腐蚀后组织清晰；
4. 晶界或相界可分辨；
5. 无严重污染；
6. 无大面积过腐蚀；
7. 图片中标尺、放大倍数和样品编号清楚；
8. 能支撑后续组织分析。

如果样品出现以下情况，应考虑重新制样或重新腐蚀：

1. 大量划痕横穿组织；
2. 腐蚀严重发黑；
3. 组织完全看不清；
4. 表面污染明显；
5. 边缘严重塌陷；
6. 关键区域被磨掉；
7. 样品方向或编号无法确认。

---

## 7. 常见错误与错因分析

## 7.1 把“磨亮”当成“制样成功”

表面发亮不代表样品合格。
金相制样的目标是让组织真实、清楚地显示出来，而不是只追求镜面效果。

## 7.2 跳砂纸太快

如果从粗砂纸直接跳到很细的砂纸，粗划痕很难去除。后续即使抛光，也可能留下深划痕。

## 7.3 每道砂纸没有磨够

如果上一道划痕没去除就进入下一道，后面会越来越难补救。

## 7.4 腐蚀时间凭感觉

不同材料、不同腐蚀剂、不同组织状态对应的腐蚀时间差异很大。腐蚀应以显微观察效果为准，不能只凭肉眼判断。

## 7.5 把制样缺陷当成材料缺陷

划痕、拖尾、第二相脱落、抛光污染、过腐蚀坑都可能被误认为材料本身缺陷。判断缺陷时必须结合形貌、位置、重复性和其他表征方法。

## 7.6 实验记录不完整

如果没有记录砂纸顺序、抛光液、腐蚀剂、腐蚀时间和样品编号，后续很难复现实验，也很难判断问题出在哪里。

## 9. 与其他知识点的关系

金相制样与后续多个知识模块直接相关：

1. 与 OM 观察相关：制样质量决定光学显微组织是否清晰；
2. 与 SEM 观察相关：表面污染、划痕和脱落会影响形貌判断；
3. 与 EDS 分析相关：污染物和残留物会干扰元素分析；
4. 与 EBSD 分析相关：表面变形层会严重影响衍射花样质量；
5. 与组织—性能分析相关：只有真实组织清晰，才能讨论性能变化原因；
6. 与失效分析相关：制样缺陷和真实裂纹必须区分开。

可以说，金相制样是材料显微组织分析的基础入口。

---

## 10. 我需要掌握到什么程度

作为本科生科研训练入门阶段，需要达到以下程度：

### Level 1：知道流程

能够说出：

```text
切割 → 镶嵌 → 磨样 → 抛光 → 腐蚀 → 观察
```

并理解每一步的目的。

### Level 2：能按流程操作

能够在学长或老师指导下完成基础金相样品制备，并记录关键参数。

### Level 3：能判断问题

看到划痕、拖尾、过腐蚀、欠腐蚀、污染等问题时，能初步判断可能原因。

### Level 4：能服务科研分析

能够把制样质量与组织观察结果联系起来，避免把制样假象误判为材料真实组织或缺陷。

当前阶段至少应达到 Level 2，并逐步向 Level 3 过渡。
