# 学习路线图

欢迎来到“金属材料科研训练知识库”的学习路线模块。

这个知识库不是为了简单堆放知识点，而是帮助本科生从零散课程知识逐步过渡到真实金属材料科研训练。它强调的不是“看过多少内容”，而是能不能建立下面这条主线：

```text
材料基础
→ 实验技能
→ 表征方法
→ 组织—性能分析
→ 文献阅读与科研写作
→ 数据处理
→ 课题组实践
→ 复习与面试表达
```

如果你刚开始接触金属材料科研训练，可以先从本页了解整体结构，再进入详细学习路线。

---

## 1. 这个模块解决什么问题

很多初学者进入课题组前会遇到几个问题：

* 不知道金属材料科研到底在研究什么；
* 不知道该先学晶体结构、实验技能，还是先学 SEM、XRD；
* 学了很多概念，但不知道它们之间有什么关系；
* 做实验时不知道实验结果最后要怎么解释；
* 准备组会或保研面试时不知道怎样讲科研经历。

学习路线模块的作用，就是把知识库中的内容按照“先后顺序”和“能力目标”串起来。

---

## 2. 推荐学习顺序

建议按下面顺序学习：

```text
Level 0：认识材料科研训练
Level 1：建立金属材料基础概念
Level 2：掌握基础实验与表征方法
Level 3：建立组织—性能分析能力
Level 4：训练文献阅读、数据处理和科研表达
Level 5：面向课题组实践和保研面试复盘
```

完整路线请进入：

[金属材料科研训练学习路线图](./roadmap.md)

## 3. 各模块定位

### 3.1 金属材料基础

对应模块：

[金属材料基础](../01-metal-basics/)

主要解决：

```text
金属材料科研研究什么？
晶体结构、晶体缺陷、工艺—组织—性能关系是什么？
```

建议优先学习：

* 材料科学与金属材料科研训练概论；
* 晶体结构基础；
* 晶体缺陷基础；
* 工艺—组织—性能逻辑链。

---

### 3.2 实验室基础技能

对应模块：

[实验室基础技能](../02-lab-skills/)

主要解决：

```text
进入实验室后，最基础的样品制备和实验规范是什么？
```

建议优先学习：

* 金相制样基础流程。

这一模块的重点不是“机械照步骤做实验”，而是理解每一步操作如何影响最终组织观察结果。

---

### 3.3 材料表征方法

对应模块：

[材料表征方法](../03-characterization/)

主要解决：

```text
SEM、EDS、XRD 等方法分别能回答什么问题？
```

建议优先学习：

* SEM 扫描电镜入门；
* EDS 能谱分析入门；
* XRD 物相分析入门。

学习时要特别注意：不同表征方法有不同证据边界，不能用一个方法证明所有结论。

---

### 3.4 组织—性能分析

对应模块：

[组织—性能分析](../04-microstructure-property/)

主要解决：

```text
如何把实验工艺、显微组织和性能数据联系起来？
```

重点学习：

* 工艺—组织—性能逻辑链进阶版。

这一模块是从“看到现象”走向“解释机制”的关键。

---

### 3.5 文献阅读与科研写作

对应模块：

[文献阅读与科研写作](../05-literature-writing/)

主要解决：

```text
如何读材料论文、写实验记录、整理周报和准备组会？
```

建议优先学习：

* 如何阅读一篇材料论文；
* 实验记录与周报模板。

这个模块会直接影响你后续组会汇报、科研积累和保研材料整理。

---

### 3.6 数据处理与图谱分析

对应模块：

[数据处理与图谱分析](../06-data-analysis/)

主要解决：

```text
如何看懂应力—应变曲线等基础性能数据？
```

建议优先学习：

* 应力—应变曲线怎么读。

这一模块的重点是让你不要只看“强度高不高”，还要能分析屈服、塑性、加工硬化、颈缩和断裂行为。

---

### 3.7 课题组项目实践

[课题组项目实践库](../07-project-practice/)

## 4. 不同使用场景下怎么学

### 4.1 如果你刚准备进组

优先看：

1. 材料科学与金属材料科研训练概论；
2. 晶体结构基础；
3. 晶体缺陷基础；
4. 金相制样基础流程；
5. SEM 扫描电镜入门；
6. 如何阅读一篇材料论文。

目标是先建立基本科研语言和实验常识。

---

### 4.2 如果你已经开始做实验

优先看：

1. 金相制样基础流程；
2. SEM 扫描电镜入门；
3. EDS 能谱分析入门；
4. XRD 物相分析入门；
5. 实验记录与周报模板；
6. 常见误区与易错点。

目标是保证实验过程可复查、表征结果能解释、周报不写成流水账。

---

### 4.3 如果你已经有实验结果

优先看：

1. 工艺—组织—性能逻辑链；
2. 工艺—组织—性能逻辑链进阶版；
3. 应力—应变曲线怎么读；
4. 常见误区与易错点；
5. 材料科研面试常见问题。

目标是把实验结果整理成清楚的组织—性能分析。

---

### 4.4 如果你正在准备保研或面试

优先看：

1. 材料科研面试常见问题；
2. 常见误区与易错点；
3. 实验记录与周报模板；
4. 如何阅读一篇材料论文；
5. 金属材料科研训练学习路线图。

目标是能把“做过什么实验”讲成“理解了什么科研逻辑”。

## 5. 学习时的自检标准

每学完一篇文章，都问自己：

```text
我能不能用 3 句话复述这篇内容？
我知道它解决什么问题吗？
我知道它和实验、论文或组会有什么关系吗？
我能不能举一个材料科研中的例子？
我知道哪些结论不能说得太绝对吗？
```

如果只是看完页面，但无法复述和应用，说明还没有真正掌握。
