# 复习卡片与问题库

本模块用于科研训练中的复盘、纠错和面试准备。

前面的模块主要帮助你学习知识、做实验、读论文和整理周报；本模块则帮助你在关键节点前检查：

```text
我有没有犯常见错误？
我能不能把科研经历讲清楚？
我能不能回答老师可能追问的问题？
我的结论是否证据充分？
我有没有把不确定内容说得太绝对？
```

它适合在组会前、和老师沟通前、保研面试前反复查看。

---

## 1. 本模块解决什么问题

本模块主要解决两类问题：

1. 科研训练中的常见误区；
2. 材料科研面试和组会答辩中的高频问题。

目前本模块包括：

1. [常见误区与易错点](./common-mistakes.md)
2. [材料科研面试常见问题](./interview-qa.md)

这两个页面分别对应：

```text
避免犯错
+
训练表达
```

## 2. 常见误区与易错点

入口页面：

[常见误区与易错点](./common-mistakes.md)

这篇文章主要帮助你检查：

* 是否只做实验，不理解问题；
* 是否只看图，不讲机制；
* 是否只报性能，不解释组织原因；
* 是否把 SEM、EDS、XRD 的证据作用混淆；
* 是否把相关性直接写成因果；
* 是否把结论写得过满；
* 是否不保存原始数据；
* 是否把周报和组会写成流水账。

这篇文章特别适合在以下场景使用：

* 写周报前；
* 准备组会前；
* 整理实验结果前；
* 修改论文或报告前；
* 准备保研科研经历前。

## 3. 材料科研面试常见问题

入口页面：

[材料科研面试常见问题](./interview-qa.md)

## 4. 本模块的使用方法

### 4.1 组会前使用

组会前检查：

```text
我本周汇报的问题是什么？
我展示的图表能支撑什么结论？
我的机制解释是否有证据？
哪些判断还只是推测？
老师可能追问什么？
```

建议先看：

* [常见误区与易错点](./common-mistakes.md)
* [材料科研面试常见问题](./interview-qa.md)

### 4.2 写周报前使用

写周报前检查：

```text
是否有本周目标？
是否有关键结果？
是否有初步分析？
是否写清问题和下周计划？
是否保存原始数据路径？
```

建议配合学习：

[实验记录与周报模板](../05-literature-writing/weekly-report.md)

### 4.3 保研面试前使用

保研面试前至少准备：

```text
2 分钟科研经历介绍
项目研究对象
自己具体工作
使用过的方法及其作用
一个组织—性能分析例子
项目不足与下一步计划
遇到困难和收获
```

建议重点看：

[材料科研面试常见问题](./interview-qa.md)

## 5. 高频自检清单

每次准备汇报或面试前，快速检查：

* [ ] 我能否用一句话概括项目？
* [ ] 我能否说清当前研究问题？
* [ ] 我能否说明自己具体做了什么？
* [ ] 我能否解释某个表征方法的作用？
* [ ] 我能否把组织变化和性能变化联系起来？
* [ ] 我能否说明结论的证据边界？
* [ ] 我是否保存了原始数据和实验条件？
* [ ] 我是否知道下一步要验证什么？
* [ ] 我是否避免使用“完全证明”“唯一原因”等过强表述？
* [ ] 我是否准备好回答项目不足？

## 7. 学完本模块应达到什么程度

### 7.1 最低要求

至少应能做到：

1. 识别自己周报或汇报中的常见错误；
2. 避免把表征结果说得过满；
3. 知道科研结论要有证据边界；
4. 能回答几个基础面试问题；
5. 能说明自己下一步要提升什么。

### 7.2 科研训练要求

进入课题组后，应进一步做到：

1. 能用问题—方法—结果—机制—不足介绍项目；
2. 能主动指出当前证据不足；
3. 能把老师追问转化为下一步实验任务；
4. 能在组会前完成自检；
5. 能不断修正自己的科研表达。

### 7.3 保研准备要求

如果希望用于保研，应达到：

1. 能用 2 到 3 分钟讲清科研经历；
2. 能解释自己做的实验服务于什么问题；
3. 能说明实验结果和材料机制之间的关系；
4. 能坦诚讲出项目不足和改进方向；
5. 能把科研训练转化为简历、面试和个人陈述素材。
